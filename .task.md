# Fishing Catch Log PWA

## Goal
A React PWA to log fishing catches ‚Äî offline-first, no subscription, no account. Sell for $8 one-time on Gumroad.
Deploy on GitHub Pages at https://xavierkain.github.io/fishing-log

## Market
- 50M+ anglers in USA. r/Fishing 2.5M members, r/flyfishing 400K.
- Existing apps (Anglr, FishBrain) require subscriptions for key features.
- People still use Excel/Google Sheets to log catches = clear gap.
- Target: anyone who wants to log catches offline, forever, for a one-time price.

## Core Features (MVP ‚Äî 1 night)
1. **Log a catch**:
   - Species (dropdown + custom), Weight (kg/lbs toggle), Length (cm/in toggle)
   - Location (text field OR tap on map to pin), Date/Time (auto-fills to now)
   - Bait/lure used, Water conditions (clear/murky/stained), Weather (sunny/cloudy/rain/wind)
   - Photo (optional, stored in IndexedDB as base64)
   - Notes
2. **Catch log** ‚Äî list view, newest first, filterable by species/location/date range
3. **Stats dashboard**:
   - Total catches, by species breakdown, biggest catch ever (weight + photo), best month
   - Personal records per species (biggest, longest)
4. **Map view** ‚Äî show all catch pins on a Leaflet map (offline tiles or OpenStreetMap)
5. **Export** ‚Äî export to CSV or PDF summary report (good for tournaments)
6. **PWA** ‚Äî installable on phone, works fully offline, dark mode

## Tech Stack
- React 18 + TypeScript + Vite
- TailwindCSS (dark green/outdoors feel ‚Äî colors: slate-900 bg, emerald-600 accents)
- Dexie.js (IndexedDB ‚Äî catches + photos)
- Leaflet + react-leaflet (map, works with OSM tiles)
- jsPDF (export report)
- vite-plugin-pwa (service worker + manifest)
- **GitHub Pages deployment**: use `gh-pages` package
  - In vite.config.ts: set `base: '/fishing-log/'`
  - package.json scripts: `"deploy": "npm run build && gh-pages -d dist"`

## GitHub Pages Setup (IMPORTANT)
- The app MUST deploy to GitHub Pages at https://xavierkain.github.io/fishing-log
- Set `base: '/fishing-log/'` in vite.config.ts
- Add to package.json: `"homepage": "https://xavierkain.github.io/fishing-log"`
